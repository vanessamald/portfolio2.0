# Portfolio2.0

## Description
An updated personal portfolio created with React and Express.js, featuring contact form capabilities powered by Nodemailer. Additionally, the website includes a newsletter sign-up feature that utilizes MySQL for data storage. This dynamic portfolio website showcases my work, skills, and achievements in an engaging and interactive manner. Built using React, it provides a smooth and seamless user experience, allowing visitors to explore my portfolio and learn more about my capabilities. With Express.js, the portfolio incorporates server-side functionality, enabling data retrieval and integration with external APIs or databases. With its modern design, responsiveness, enhanced contact capabilities, and newsletter functionality, this portfolio serves as a comprehensive representation of my professional journey, highlighting my technical proficiency and accomplishments.
